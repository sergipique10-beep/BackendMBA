# BackendMBA

Proyecto final del módulo de **Node.js y Backend**: API REST con Express, MongoDB Atlas y Cloudinary.

```
BackendMBA/
├── backend/   ← Servidor Express + MongoDB Atlas + Cloudinary (proyecto evaluado)
└── frontend/  ← Cliente Vite + JavaScript vanilla que consume la API (no forma parte del encargo)
```

## Documentación

- **[backend/README.md](backend/README.md)** — documentación completa de la API: modelos, endpoints, reglas de negocio, autenticación y ejemplos de uso.

## Puesta en marcha rápida

```bash
# Backend (API en http://localhost:3000)
cd backend
npm install
npm run dev

# Frontend (cliente en http://localhost:5173, opcional)
cd frontend
npm install
npm run dev
```
